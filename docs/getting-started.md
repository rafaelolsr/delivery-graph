# Getting Started

## 1. Intake a demand

```text
/dge-intake add a validation gate for advisor eval regressions
```

## 2. Plan the graph

```text
/dge-plan-graph
```

## 3. Sync nodes

```text
/dge-sync linear
```

## 4. Work one node

```text
/dge-work-node NODE-001
```

## 5. Verify

```text
/dge-verify NODE-001
```

## 6. Review and compound

```text
/dge-review
/dge-compound NODE-001
```
